# MovieVerse

MovieVerse is a front-end website for displaying movie information, built using pure HTML, CSS, and JavaScript (no frameworks).

## Folder Structure

- **css/**: Contains CSS files for styling the website pages.
- **js/**: Contains JavaScript files for adding interactivity and functionality.
- **assets/posters/**: Directory for storing movie poster images.
- **data/**: Directory for data files, such as JSON files containing movie information or reviews.

## Files

- **index.html**: The home page of the website, providing an overview and navigation to other sections.
- **movies.html**: Page that lists all available movies.
- **movie.html**: Template or page for displaying individual movie details.
- **reviews.html**: Page for viewing and managing movie reviews.
- **about.html**: About page with information about the website or project.
- **privacy.html**: Privacy policy page.
- **contact.html**: Contact page for user inquiries.
- **README.md**: This file, describing the project structure and purpose.

## Getting Started

To view the website, open `index.html` in a web browser. Ensure all files are in the same directory structure as described.

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
